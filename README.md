Batshield
=========

An audio shield for arduino intended to develop audio synthesis and filtering.

Features
========

- MIDI input
- MIDI output
- Mic/Line input
- Line output
- 2 PWM output
- 4 switches
- 4 push buttons
- 5 knobs


The line output have been designed in order to be compatible with mozzi library
(http://sensorium.github.io/Mozzi/)
